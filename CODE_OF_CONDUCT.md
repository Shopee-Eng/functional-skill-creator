# Code of Conduct

## Basic Standards

We want this project to be a respectful, pragmatic, and collaborative place. Please be kind, direct, and constructive. Default to good faith, but take responsibility for impact.

Unacceptable behaviors include harassment, personal attacks, discriminatory language, publishing others' private information, and persistently disrupting project work.

## Enforcement

Maintainers may delete comments, close issues, or restrict participation that harms the project community.

To report a Code of Conduct issue, open a private conversation with the repository maintainers via GitHub, or use the contact path listed in [SECURITY.md](SECURITY.md) if you need a non-public channel.
